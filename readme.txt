Схема поиска значений параметров расписания:
1. Сначала перебор всех значений r_rf (начиная с 0.99)
2. Внутри предыдущего цикла перебираем все длины цепочки r_mcc (начиная с 1)
3. Если построено расписание с параметрами r_rf и r_mcc, то это и будет ответом.
4. Иначе расписание построить не удалось.


Сборка:
1. Зайти в папку с проектом
2. mkdir build
3. cd build
4. cmake ..
5. make all


Пример запуска программы
1. Зайти в папку с проектом
2. ./prog_02160506_521 ../data/S1_SHIFT_20ms_025_055_021.txt 20
